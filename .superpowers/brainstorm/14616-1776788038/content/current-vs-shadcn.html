关键差异总结

当前设计
自定义 CSS 变量（--bg-primary, --accent-primary 等）
紫色强调色贯穿全局
渐变背景光效
紧凑、精致的间距
细线分割 + 居中标签
主题切换已内置
shadcn-vue 风格
Tailwind CSS 工具类 + CSS 变量（--background, --foreground 等）
中性色调，可通过主题自定义
平面化、干净的卡片设计
更大更统一的圆角
标题 + 内容区块分组
暗色模式通过 class 切换
